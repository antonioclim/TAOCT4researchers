A retry policy may be described as follows:

```text
attempt := 0
while attempt < K:
  response := GET(url)
  if 200 <= response.status < 300:
     return response
  if response.status in {429, 503}:
     sleep(backoff(attempt))
     attempt := attempt + 1
     continue
  raise PermanentError
raise RetryExhausted
```
