# Pacing Guide

## The Art of Computational Thinking for Researchers

**Extended Edition — Fourteen Instructional Units**

---

## Overview

This guide provides detailed time allocation recommendations for each unit, supporting both self-directed learners and instructors planning course delivery. The estimates assume focused engagement without interruption; actual completion times vary according to prior experience and individual learning pace.

## Adjustment Factors

Modify estimates based on participant profile:

| Factor | Adjustment |
|--------|------------|
| No prior Python | +30% |
| Strong CS background | −20% |
| Mathematical preparation | −10% for Units 03, 05, 08 |
| Database experience | −20% for Unit 10 |
| Web development background | −25% for Unit 12 |
| Statistics/ML exposure | −20% for Unit 13 |

## Progress Tracking

Each unit includes a progress checklist in its README.md. Participants should:

1. Complete all checklist items before advancing
2. Achieve ≥70% on quizzes before proceeding
3. Ensure all laboratory tests pass
4. Review self-assessment reflections periodically
