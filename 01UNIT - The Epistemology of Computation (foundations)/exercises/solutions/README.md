# Exercise Solutions

Solutions for the practice exercises are provided in the `lab/solutions/` directory.

## Solution Files

| Exercise Level | Topics Covered | Solution Location |
|---------------|----------------|-------------------|
| Easy 01-03 | TM traces, components, lambda syntax | `lab/solutions/lab_1_01_solutions.py` |
| Medium 01-03 | TM design, beta reduction, AST construction | `lab/solutions/lab_1_02_solutions.py` |
| Hard 01-03 | Multi-tape TM, Church predecessor, letrec | `lab/solutions/lab_1_03_solutions.py` |

## Using Solutions

1. Attempt the exercise first without looking at solutions
2. If stuck, review the hints in the exercise file
3. Compare your approach with the solution
4. Note any differences in style or efficiency

## Code Standards

All solutions follow the course coding standards:

- British English in comments and documentation
- Type hints on all function signatures
- Google-style docstrings
- Logging instead of print statements
- Comprehensive error handling
