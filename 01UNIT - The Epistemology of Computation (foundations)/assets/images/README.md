# Week 1 Images

This directory contains supporting images for the course materials.

## SVG Diagrams

The primary visual assets are in `../diagrams/`:

- `turing_machine_architecture.svg` — TM components visualisation
- `lambda_calculus_syntax.svg` — Lambda expression syntax tree
- `ast_structure.svg` — AST node hierarchy

## Adding Images

When adding images:

1. Use SVG format for diagrams (scalable, editable)
2. Use PNG for screenshots or raster graphics
3. Follow naming convention: `topic_description.ext`
4. Include alt text descriptions in accompanying markdown
