13UNIT — Machine Learning for Researchers

Slides (HTML) for lecture delivery and self-study.

1. Evaluation as an experimental design

Generalisation is assessed under a protocol. A score without a protocol is not evidence.

2. Leakage and pipelines

Fit preprocessing on training data only. Use Pipeline to enforce this constraint.

3. Bias–variance diagnostics

Learning curves help decide whether more data or a different model class is likely to help.

4. Unsupervised exploration

Clustering is descriptive. Validate stability and interpretability rather than searching for a single “true” partition.

5. Reporting

Report metrics, variability, assumptions and failure modes. Record seeds and configuration.
