# Sample Datasets

This directory contains sample datasets for use in laboratory exercises and practice problems.

## Available Datasets

| File | Description | Format | Size |
|------|-------------|--------|------|
| `sensor_readings.json` | Environmental sensor measurements | JSON | ~2KB |
| `experiment_log.csv` | Experiment metadata records | CSV | ~1KB |

## Usage

These datasets are designed for educational purposes and demonstrate common data structures encountered in research computing:

- **sensor_readings.json**: Nested JSON structure with metadata and time-series measurements
- **experiment_log.csv**: Tabular data with mixed types (strings, numbers, dates)

## Data Generation

Sample datasets are synthetic and generated for educational purposes. They do not represent actual research data.
